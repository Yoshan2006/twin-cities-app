# Twin Cities Web Application

## 📌 Project Overview
The **Twin Cities App** is a data-driven web application designed to provide users with real-time weather updates, places of interest, and travel routing between two selected cities (e.g., Liverpool and Dublin). This project demonstrates the practical application of back-end API integrations and secure database management.

## 🚀 Features
* **Real-time Weather Data:** Integrated with the OpenWeather API to fetch live weather conditions.
* **Dynamic Points of Interest:** Utilizes the Flickr API / custom data to display dynamic location-based images and details.
* **Secure Database Handling:** Implements PHP Data Objects (PDO) for secure, object-oriented database interactions.
* **Robust Error Handling:** Features custom global exception and error handlers to prevent sensitive data leaks and ensure a seamless user experience.

## 💻 Tech Stack
* **Back-end:** PHP 8, OOP (Object-Oriented Programming)
* **Database:** MySQL, PDO
* **Front-end:** HTML5, CSS3, JavaScript
* **APIs:** OpenWeather API, Flickr API

## ⚙️ Setup Instructions
1. Clone the repository to your local machine.
2. Import the `twin_cities_db.sql` file into your local MySQL server.
3. Update the `config.php` file with your local database credentials and API keys.
4. Run the application using a local server environment like XAMPP or MAMP.
